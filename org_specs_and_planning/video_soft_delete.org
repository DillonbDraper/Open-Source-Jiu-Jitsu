* The goal here is to create a soft delete system for videos that preserves user notes/messages referencing videos while removing them from the system
** We should begin by adding a deleted_at (timestamp) column on the videos resource
** Queries for videos in both the video dashboard and technique tree should filter out soft deleted videos
** The utility on the user_profile page that currently hard deletes videos should now instead populate this timestamp.
** Currently, messages and notes may reference videos, which may be deleted.  When a video is deleted, it should trigger a system message to all users with messages/notes referencing a video explaining that that video has been deleted by contributor.
*** Links to disabled videos should be disabled in those messages/notes.
*** Notes should have a toggle by the table that enables users to toggle on/off showing notes for deleted videos.  It should be off by default.
